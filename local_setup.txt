Local Setup Instructions for WMS Application:

1. Install required Python packages:
   pip install Flask==3.0.0
   pip install Flask-SQLAlchemy==3.1.1
   pip install Flask-Login==0.6.3
   pip install Werkzeug==3.0.1
   pip install Jinja2==3.1.2
   pip install SQLAlchemy==2.0.23
   pip install email-validator==2.1.0
   pip install requests==2.31.0
   pip install gunicorn==21.2.0
   pip install psycopg2-binary==2.9.9
   pip install PyJWT==2.8.0

2. Set environment variables (optional):
   - SESSION_SECRET=your-secret-key-here
   - DATABASE_URL=sqlite:///wms.db (or your PostgreSQL URL)
   - SAP_B1_SERVER=your-sap-server-url
   - SAP_B1_USERNAME=your-sap-username
   - SAP_B1_PASSWORD=your-sap-password
   - SAP_B1_COMPANY_DB=your-company-database

3. Run the application:
   python main.py

4. Access the application at:
   http://localhost:5000

5. Default login credentials:
   Username: admin
   Password: admin123

Note: The application will create a local SQLite database if no DATABASE_URL is provided.
